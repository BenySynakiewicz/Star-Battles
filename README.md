# Star Battles 1.1

A very simple clone of the [Space Invaders](https://en.wikipedia.org/wiki/Space_Invaders), created for the purpose of learning the basics of the [pygame](https://www.pygame.org/) framework.

## Dependencies

- **pygame** (pygame).
- **NumPy** (numpy).
